Features

Everything you need for production backends

A complete application framework built on Web Standard APIs. Task queues, cron, multi-runtime — not just routing.

🧩

Plugin Encapsulation on Web Standards

Fastify-style scoped plugins built on Request/Response, not req/res. Hooks and decorations never leak across boundaries. Works on every runtime.

⏰

Background Tasks & Cron Built In

Job queues with retry/backoff and cron scheduling out of the box. No external packages needed. Swap to Redis for production with one line.

🔍

Schema-Agnostic Validation

Auto-detects Zod, TypeBox, or Valibot. No config, no adapters. Just pass your schema and get validated input with structured error responses.

🔌

RPC + OpenAPI Generation

Type-safe procedures with input/output validation, compose routers, call from the client with full TypeScript inference. OpenAPI 3.1 and Swagger UI generated automatically.

🛡

Security & Middleware First-Party

CSRF protection, rate limiting, JWT, compression, CORS, and Helmet-style security headers. All built in. No hunting for third-party middleware.

Type-Safe RPC

Build procedures with schema validation and auto-generated clients

import { procedure, router } from '@celsian/rpc';
import { z } from 'zod';

const appRouter = router({
  user: {
    getById: procedure
      .input(z.object({ id: z.string() }))
      .query(async ({ input }) => {
        return db.users.find(input.id);
      }),
    create: procedure
      .input(z.object({ name: z.string() }))
      .mutation(async ({ input }) => {
        return db.users.create(input);
      }),
  },
});

// Client — full type inference
const user = await rpc.user.getById.query({ id: '1' });

Server-Sent Events

Real-time streaming with fan-out hub pattern

import { createSSEStream, createSSEHub }
  from '@celsian/core';

// Single client stream
app.get('/api/events', (req) => {
  const channel = createSSEStream(req);
  channel.send({ event: 'hello', data: { ok: true } });
  return channel.response;
});

// Fan-out to all subscribers
const hub = createSSEHub();

app.get('/api/live', (req) => {
  const channel = hub.subscribe(req);
  return channel.response;
});

// Broadcast to everyone
hub.broadcast({ event: 'update', data: payload });

Background Tasks

Job queue with retry, concurrency, and cron scheduling

import { createTaskQueue, createCronScheduler }
  from '@celsian/core';

const queue = createTaskQueue({ concurrency: 5 });

queue.register({
  name: 'send-email',
  handler: async (job) => {
    await sendEmail(job.payload);
    return { success: true };
  },
  maxRetries: 3,
  retryDelayMs: 5000,
});

// Enqueue with optional delay
await queue.enqueue('send-email', { to: 'ada@example.com' });

// Cron jobs with human-readable schedules
const cron = createCronScheduler();
cron.add({ name: 'cleanup', schedule: 'every 1h',
  handler: () => db.cleanup() });

Caching & Sessions

Response cache, KV store, and session management

import { MemoryKVStore, createResponseCache,
  createSessionManager } from '@celsian/cache';

const store = new MemoryKVStore();
const cache = createResponseCache({ store, ttlMs: 60_000 });
const sessions = createSessionManager({ store });

// Automatic response caching
app.get('/api/data', (req, reply) => {
  return cache.cached(req, () => {
    return reply.json(expensiveQuery());
  });
}); // x-cache: HIT | MISS

// Session management
app.post('/api/login', async (req, reply) => {
  const session = await sessions.create({ user });
  return reply.header('set-cookie', sessions.cookie(session.id))
    .json({ ok: true });
});

Get Started

Ship your API in minutes

Scaffold a new project, add your routes, deploy anywhere.

$ npx create-celsian my-api
  • 1 Scaffold your project with npx create-celsian my-api
  • 2 Define routes, RPC procedures, and middleware in your app
  • 3 Run celsian dev for development with hot reload
  • 4 Deploy with Bun.serve({ fetch: app.fetch }) or any adapter
server.ts 
import { createApp, cors, jwtAuth } from '@celsian/core';
import { procedure, router } from '@celsian/rpc';
import { z } from 'zod';

const app = createApp();
await app.register(cors, { origin: '*' });

// REST routes
app.get('/health', (req, reply) =>
  reply.json({ status: 'ok' })
);

// Type-safe RPC
const rpc = router({
  hello: procedure
    .input(z.object({ name: z.string() }))
    .query(({ input }) => `Hello ${input.name}!`),
});

app.mount('/rpc', rpc);

// One-line deploy
Bun.serve({ fetch: app.fetch, port: 3000 });
